# 实现计划: 中文界面本地化

## 概述

本实现计划将 QueryWeaver 前端应用的界面本地化为中文，支持中英文切换。实现基于 react-i18next 国际化框架，包括界面文本翻译、日期时间格式化、数字格式化以及语言切换功能。
