# 实现计划：达梦和人大金仓数据库支持

## 概述

本实现计划将为 QueryWeaver 项目添加达梦数据库和人大金仓数据库的支持。实现将遵循现有的 BaseLoader 架构，创建两个新的数据库加载器类，并更新相关的路由和配置。
