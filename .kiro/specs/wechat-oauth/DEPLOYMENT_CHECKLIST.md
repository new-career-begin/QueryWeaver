# 微信和企业微信 OAuth 登录部署检查清单

本文档提供了部署微信和企业微信 OAuth 登录功能的完整检查清单，确保所有必需的配置和测试都已完成。

---

## 1. 前置条件检查

### 1.1 域名和证书
- [ ] 已有已备案的域名（中国大陆必需）
- [ ] 已配置 HTTPS 证书
- [ ] 证书有效期充足（建议 > 30 天）
- [ ] 域名可以正常访问

### 1.2 微信开放平台配置
- [ ] 已注册微信开放平台账号 (https://open.weixin.qq.com/)
- [ ] 已创建网站应用
- [ ] 已配置授权回调域名
- [ ] 应用已通过审核
- [ ] 已获取 AppID
- [ ] 已获取 AppSecret

### 1.3 企业微信配置
- [ ] 已注册企业微信账号 (https://work.weixin.qq.com/)
- [ ] 已创建自建应用或第三方应用
- [ ] 已配置可信域名
- [ ] 已获取 CorpID（企业ID）
- [ ] 已获取 AgentID（应用ID）
- [ ] 已获取 CorpSecret（应用密钥）

## 2. 环境变量配置

### 2.1 必需的环境变量
- [ ] `FASTAPI_SECRET_KEY` - FastAPI 会话密钥（必需）
- [ ] `FALKORDB_URL` - FalkorDB 连接 URL（必需）

### 2.2 微信登录配置
- [ ] `WECHAT_APP_ID` - 微信应用 ID
- [ ] `WECHAT_APP_SECRET` - 微信应用密钥

### 2.3 企业微信登录配置
- [ ] `WECOM_CORP_ID` - 企业微信企业 ID
- [ ] `WECOM_AGENT_ID` - 企业微信应用 ID
- [ ] `WECOM_CORP_SECRET` - 企业微信应用密钥

### 2.4 可选配置
- [ ] `OAUTH_BASE_URL` - OAuth 回调基础 URL（如果需要覆盖）
- [ ] `OAUTHLIB_INSECURE_TRANSPORT` - 开发环境 HTTP 支持（仅开发环境）

### 2.5 配置验证
```bash
# 检查环境变量是否已设置
echo $WECHAT_APP_ID
echo $WECHAT_APP_SECRET
echo $WECOM_CORP_ID
echo $WECOM_AGENT_ID
echo $WECOM_CORP_SECRET
```

## 5. 安全检查

### 5.1 CSRF 防护
- [ ] 授权请求包含 state 参数
- [ ] 回调处理验证 state 参数
- [ ] state 参数有时效性（10 分钟）
- [ ] state 参数使用加密签名

### 5.2 Token 安全
- [ ] API Token 使用 `secrets.token_urlsafe(32)` 生成
- [ ] Cookie 设置 `httponly=True`
- [ ] Cookie 设置 `secure=True`（生产环境）
- [ ] Token 过期时间为 24 小时
- [ ] 敏感配置从环境变量读取

### 5.3 数据安全
- [ ] AppSecret 和 CorpSecret 不在代码中硬编码

## 13. 故障排查

### 13.1 常见问题
| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 无法跳转到授权页面 | 配置未启用 | 检查环境变量是否设置 |
| 授权后回调失败 | 回调 URL 配置错误 | 检查微信/企业微信后台配置 |
| 用户信息获取失败 | Access Token 无效 | 检查 AppSecret/CorpSecret |
| 登录后无法访问资源 | Cookie 未设置 | 检查 Cookie 配置 |
| CSRF 验证失败 | State 参数不匹配 | 检查 state 生成和验证逻辑 |

### 13.2 调试步骤
1. 检查应用日志
2. 检查浏览器控制台
3. 检查网络请求
4. 检查环境变量配置
5. 检查微信/企业微信后台配置
